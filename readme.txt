== Description ==
 
This is a plugin for managing WordPress Contact Form Messages. Incoming messages are stored in the WordPress DB in order to manage and reply message directly from the WordPress backend. In addition the administrator receives a copy by email plus if desired the sender receives a customizable Auto Response Message. 

In addition the plugin comes with a simple mail statistic and a user permission functionality. 
 
== Installation ==
 
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Copy the following shortcode [easy_contact] into your contact page
